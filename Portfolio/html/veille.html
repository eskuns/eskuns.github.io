Sujet : Python et la Big Data

Au cours des dernières années, Python est devenu un langage de programmation incontournable dans le domaine de la big data, grâce à sa simplicité, sa polyvalence et son vaste écosystème de bibliothèques spécialisées.

Si Python sort du lot dans la Data Science c'est pour plusieurs raisons, comme sa simplicité ou encore la richesse du langage et des bibliothèques qui compose celui-ci, de plus il propose un grand nombre de framework destiné à la Data Science. Python est utilisé pour de analyse de données, des prédictions/modélisations ou encore du traitement de langage naturel comme nous en parle l'article d'ETIC INSA publié en 2021.

Python durant le Lycée :
La première fois que j'ai entendu parler de ce langage était en classe de première dans ma spécialité NSI. C'est d'ailleurs l'une des seules compétences hors du HTML/CSS que j'ai acquérit dans mes années lycées, néanmoins c'était assez complet pour les besoins de cette époque.

Python durant mes années de BTS SIO SLAM :
même si je n'ai presque pas utilisé Python durant ces deux années si ce n'est dans quelques évaluations, j'ai essayé de rester informé à propos de Python afin de pourquoi pas pllus tard me spécialiser dans ce langage étant donné qu'il me plaît et que ce qu'il permet me plaît tout autant. Python est très accessible et il existe une grande quantité d'informations concernant comment l'utiliser. Lors de ces deux années, Python a su se démarquer avec l'intégration des technologies cloud, le machine learning ou encore l'open source de son écosystème.
